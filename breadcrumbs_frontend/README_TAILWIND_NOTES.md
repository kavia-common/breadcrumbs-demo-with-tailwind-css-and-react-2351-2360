# Breadcrumbs Demo with Tailwind

This app demonstrates an accessible Breadcrumbs component with Tailwind CSS and a minimal Ocean Professional theme.

How to run:
- npm install
- npm start

How to test:
- npm test

Notes:
- Hash-based navigation is used to persist state across reloads.
- The footer shows whether common env vars are set (masked to first 4 chars).
- Tailwind is configured via tailwind.config.js, postcss.config.js, and directives in src/index.css.
